# Uniflow

Uniflow is a free and open source keyboard and mouse sharing app. Use the keyboard, mouse, or trackpad of one computer to control nearby computers, and work seamlessly between them.

Uniflow is based on Deskflow, with optimizations and enhancements for the Uniflow distribution.

[Homepage](https://uniflow.org) [Code](https://github.com/uniflow/uniflow)
